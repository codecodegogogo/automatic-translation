# 熬夜做的翻译

这是一个油猴用户脚本，用来按设定语言方向自动翻译网页内容。脚本默认会翻译所有网站，也可以按网站单独关闭或开启，并支持更换翻译服务。

脚本文件：`熬夜做的翻译.txt`

## 主要功能

- 默认按“英语 -> 简体中文”翻译网页内容。
- 可在设置页修改翻译方向，例如“所有语言 -> 简体中文”“简体中文 -> 法语”。
- 可在设置页切换全局默认行为：默认翻译或默认不翻译。
- 每个网站可以单独设置为“翻译”或“不翻译”。
- 页面右上角显示“译”悬浮图标，可拖动到合适位置。
- “译”图标 5 秒无操作后会自动淡化，默认不透明度为 50%。
- 第一次安装后打开网页时，“译”图标会向页面内侧发出 5 秒明显广播特效，并显示查看默认设置提示。
- 点击“译”图标可打开当前网站的翻译开关。
- 弹窗中的齿轮按钮可打开设置页面。
- 可选择是否每个网页都自动弹出“是否翻译”提示。
- 设置页面可搜索、选择并修改手动操作过的网站。
- 设置页里的下拉列表最多显示 6 项，更多选项可滚动查看。
- 支持微软翻译、Google 翻译、MyMemory 翻译和自定义翻译接口。
- 会监听页面变化、滚动加载内容，并尝试补翻新出现的文字。

## 安装方法

1. 在浏览器安装用户脚本管理器，例如 Tampermonkey、Violentmonkey 或 Greasemonkey。
2. 新建用户脚本。
3. 将 `熬夜做的翻译.txt` 中的全部内容复制进去。
4. 保存脚本并启用。
5. 打开任意网页，右上角会出现“译”图标。

## 软件使用说明

### 第一次使用

脚本启用后，打开任意网页会在右上角看到“译”按钮。第一次安装后，按钮附近会出现 5 秒提示特效，建议先点击按钮，再点击齿轮进入设置页，确认默认翻译行为、翻译方向和翻译服务。

默认设置是：

- 默认自动翻译。
- 翻译方向为“英语 -> 简体中文”。
- 翻译服务为“微软翻译”。
- “译”按钮 5 秒无操作后变为 50% 不透明。
- “每个网页都提示是否翻译”默认关闭。

### 日常翻译操作

打开网页后，如果当前网站允许翻译，脚本会自动处理页面文字。页面滚动、加载新内容或动态变化时，脚本会尝试继续补翻新出现的文字。

如果想临时修改当前网站：

1. 点击网页右上角“译”按钮。
2. 在弹窗中选择“翻译”或“不翻译”。
3. 选择会按当前网站保存，下次打开同一网站会继续使用这个选择。

### 移动按钮位置

按住“译”按钮拖动到合适位置，松开后位置会自动保存。以后打开网页时会继续使用这个位置。

### 修改默认行为

点击“译”按钮，再点击齿轮进入设置页。在“默认行为”里可以修改“默认是否自动翻译”：

- 开启：没有手动设置过的网站都会自动翻译。
- 关闭：没有手动设置过的网站不会自动翻译，只有手动选为“翻译”的网站才会翻译。

旁边的“图标透明度”可以设置按钮淡化后的不透明度。比如 `50` 表示淡化到 50%，`100` 表示不淡化。

“每个网页都提示是否翻译”默认关闭。开启后，每次进入网页都会自动展开右上角“翻译此网站？”弹窗，方便你逐个网站选择“翻译”或“不翻译”。

### 设置翻译方向

在设置页“翻译服务”区域，可以设置来源语言和目标语言。常用设置示例：

- 英语 -> 简体中文
- 所有语言 -> 简体中文
- 简体中文 -> 法语

选择“所有语言”时，脚本会尽量让翻译接口自动识别来源语言。

### 管理手动操作过的网站

设置页里的“手动操作过的网站”只显示你手动选择过“翻译”或“不翻译”的网站。自动跟随默认设置的网站不会显示。

使用方式：

1. 点击放大镜可以展开或隐藏搜索框。
2. 输入关键字筛选网站。
3. 在网站下拉列表中选择目标网站。
4. 使用“是否翻译”开关修改这个网站的选择。

### 切换翻译服务

在设置页“服务”下拉列表中选择翻译服务。当前内置：

- 微软翻译
- Google 翻译
- MyMemory 翻译
- 自定义接口

切换后会自动保存，并清空旧翻译缓存。后续翻译会使用新的服务。

### 使用自定义接口

选择“自定义接口”后，需要填写接口地址和结果字段。接口地址可以使用 `{text}`、`{from}`、`{to}` 占位符。适合简单 GET 接口，不适合需要动态签名的正式 API。

### 遇到问题时

- 如果网页没有翻译，先确认当前网站是否被设置为“不翻译”。
- 如果设置不生效，刷新网页再试。
- 如果微软翻译不可用，可以切换到 Google、MyMemory 或自定义接口。
- 如果某些文字没有翻译，通常是网页动态加载或特殊结构导致，滚动页面或等待片刻可能会补翻。

## 基本使用

### 默认翻译

脚本安装启用后，默认会翻译打开的网站。可以在设置页把全局默认行为改为“不翻译”。没有手动操作过的网站不会出现在设置页的网站列表里。

第一次安装后打开网页时，“译”图标会向页面内侧发出 5 秒明显广播特效，并显示“建议先查看默认设置”提示。

### 关闭当前网站翻译

1. 点击页面右上角的“译”图标。
2. 在弹窗中点击“不翻译”。
3. 当前网站会被记住，以后打开这个网站默认不翻译。

### 重新开启当前网站翻译

1. 点击“译”图标。
2. 在弹窗中点击“翻译”。
3. 当前网站会被记住，以后打开这个网站默认翻译。

### 移动图标位置

按住右上角的“译”图标并拖动到想放的位置，松开后位置会自动保存。下次打开网页时会使用上次的位置。

### 图标自动淡化

“译”图标在 5 秒无操作后会自动淡化。选择“翻译”或“不翻译”后，也会立即淡化。默认淡化后不透明度为 50%。

打开弹窗、拖动图标或鼠标悬停在图标上时，图标会恢复清晰显示。

## 设置页面

点击“译”图标后，再点击弹窗右上角的齿轮按钮，可以打开设置页面。

设置页面主要包含这些内容：

### 默认行为

这里可以用开关选择“默认是否自动翻译”。

- 开启：没有手动设置过的网站会自动翻译。
- 关闭：没有手动设置过的网站不会自动翻译，只有手动选为“翻译”的网站才会翻译。

也可以开启“每个网页都提示是否翻译”。开启后，打开网页时会自动弹出右上角提示；关闭后，需要手动点击“译”按钮才会打开提示弹窗。

### 翻译服务

默认使用“微软翻译”。可以切换为：

- 微软翻译
- Google 翻译
- MyMemory 翻译
- 自定义接口

切换翻译服务后，脚本会清空当前翻译缓存，并使用新的服务重新翻译后续内容。设置页里的选项会自动保存，不需要点击保存按钮。

### 默认微软接口说明

脚本默认使用的“微软翻译”不是你自己申请的 Azure Translator 正式密钥接口，而是接近 Edge 浏览器网页翻译功能的接口链路。

脚本会先请求临时授权：

```text
https://edge.microsoft.com/translate/auth
```

然后用拿到的临时 token 请求翻译接口：

```text
https://api-edge.cognitive.microsofttranslator.com/translate
```

这个 token 不是你的微软账户 token，也不是本机 Edge 浏览器登录账户的 token。脚本不会读取你的微软账号、浏览器账号或 Cookie。它更像是网页翻译功能临时发放的访问票据，有效期较短，失效后脚本会重新获取。

这套接口已经随着 Edge 网页翻译功能实际使用了多年，但它不是微软正式发布给第三方脚本使用的公开 API，没有官方长期兼容承诺。微软以后可能调整 token、请求头、限流策略或接口规则。

如果默认微软接口失效，可以先切换到脚本内置的 Google 翻译、MyMemory 翻译，或者使用下面的自定义接口。长期稳定的方案是接入 Azure Translator、有道智云、百度翻译、腾讯翻译等正式 API，但这些通常需要 app key、secret 和动态签名，当前脚本的自定义接口只支持简单 GET 请求，不能直接生成签名。

### 翻译方向

默认方向是“英语 -> 简体中文”。可以在设置页里选择：

- 来源语言：英语、简体中文、法语等，也可以选“所有语言”自动识别。
- 目标语言：简体中文、法语、英语等具体语言。

示例：

- `英语 -> 简体中文`
- `简体中文 -> 法语`
- `所有语言 -> 简体中文`

选择后会直接改变翻译接口的来源语言和目标语言参数。

### 图标显示

图标显示设置放在“翻译服务”区域里。可以修改“淡化后不透明度”，填写范围建议为 10 到 100：

- `50` 表示淡化后显示为 50% 不透明。
- `100` 表示不淡化。
- 数值越小，图标越透明。

### 手动操作过的网站

这里只显示你手动点过“翻译”或“不翻译”的网站。自动跟随默认行为的网站不会显示在这里。

你可以在这里：

- 点击放大镜展开搜索框，再输入关键字搜索网站。
- 从下拉列表中选择网站。
- 用“是否翻译”开关修改选中网站。

当前版本不再提供单独的“默认”按钮。需要改变某个网站时，直接在下拉列表里选择它并修改翻译选择。

## 自定义翻译接口

在设置页面选择“自定义接口”后，可以填写接口地址和结果字段。

接口地址支持这些占位符：

- `{text}`：要翻译的文本。
- `{from}`：来源语言，无法判断时为 `auto`。
- `{to}`：目标语言，默认是简体中文。

示例：

```text
https://example.com/translate?q={text}&from={from}&to={to}
```

如果接口返回 JSON，可以在“结果字段”里填写翻译结果所在字段，例如：

```text
translatedText
```

也支持简单的点路径：

```text
data.text
```

如果接口返回的不是 JSON，脚本会把接口返回的文本直接当作翻译结果。

### 自定义接口示例

MyMemory 示例，比较适合当前脚本的简单 GET 格式：

接口地址：

```text
https://api.mymemory.translated.net/get?q={text}&langpair={from}|{to}
```

结果字段：

```text
responseData.translatedText
```

有道/网易网页接口示例：

接口地址：

```text
https://fanyi.youdao.com/translate?doctype=json&type=AUTO&i={text}
```

结果字段：

```text
translateResult.0.0.tgt
```

如果只想固定“英语 -> 简体中文”，有道接口地址也可以写成：

```text
https://fanyi.youdao.com/translate?doctype=json&type=EN2ZH_CN&i={text}
```

有道这个地址属于网页接口，不是正式开放平台接口，可能会限流或失效。正式的有道智云接口需要签名，当前脚本不能直接填写使用。

## 注意事项

- 翻译内容会发送到你选择的翻译服务，请不要在敏感页面开启翻译。
- 自定义接口目前按 GET 请求调用。
- 默认微软接口使用的是接近 Edge 网页翻译的临时 token 链路，不读取你的微软账号，但不属于正式公开 API。
- 油猴脚本可以添加部分普通请求头，但不能可靠完整伪装成 Edge 浏览器。长期稳定性不能依赖请求头伪装。
- 某些网站有复杂的动态页面结构，可能会出现少量文字未翻译或重复补翻。
- 如果网站本身禁止脚本运行，油猴脚本可能无法生效。
- MyMemory 等免费接口可能有频率限制，翻译大量文字时速度可能较慢。

## 常见问题

### 为什么有些网站没有出现在设置页？

设置页只显示手动操作过的网站。默认自动翻译的网站不会显示。

### 关闭翻译后还能恢复吗？

可以。点击“译”图标后选择“翻译”，或在设置页中把该网站改为“翻译”。

### 如何回到默认状态？

进入设置页，调整“默认是否自动翻译”开关。已经手动设置过的网站会继续使用自己的选择。

### 更换翻译服务后之前的结果会怎样？

脚本会清空翻译缓存，并使用新服务处理之后的翻译。已经显示在页面上的翻译内容会尽量恢复后再重新翻译。

### 默认微软接口和 Edge 浏览器一样吗？

它和 Edge 浏览器内置网页翻译使用的是同类链路：先从 `edge.microsoft.com/translate/auth` 获取临时 token，再请求 `api-edge.cognitive.microsofttranslator.com/translate` 翻译。脚本只是复用了这条核心请求方式，不等于微软把它作为正式 API 开放给脚本长期调用。

### 这个微软 token 是谁的？

不是你的微软账户 token，也不是本机 Edge 登录账户的 token。脚本不会检测你登录了哪个微软账号。它通常是短期可用的匿名/公共翻译访问 token，用来调用网页翻译接口。

### 微软接口失效怎么办？

先在设置页切换到 Google 翻译或 MyMemory 翻译。也可以选择“自定义接口”，填入上面的 MyMemory 或有道示例。如果需要长期稳定和可控额度，建议后续改脚本接入正式翻译 API。
