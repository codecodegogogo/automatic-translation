# 无感自动翻译

这是一个油猴用户脚本，用来按设定语言方向自动翻译网页内容。脚本默认会翻译所有网站，也可以按网站单独关闭或开启，并支持更换翻译服务。

脚本文件：`自动翻译.txt`

## 主要功能

- 默认按“英语 -> 简体中文”翻译网页内容。
- 可在设置页修改翻译方向，例如“所有语言 -> 简体中文”“简体中文 -> 法语”。
- 可在设置页切换全局默认行为：默认翻译或默认不翻译。
- 每个网站可以单独设置为“翻译”或“不翻译”。
- 页面右上角显示“译”悬浮图标，可拖动到合适位置。
- “译”图标 5 秒无操作后会自动淡化，默认不透明度为 50%。
- 第一次安装后打开网页时，“译”图标会闪烁 3 秒，提示先查看默认设置。
- 点击“译”图标可打开当前网站的翻译开关。
- 弹窗中的齿轮按钮可打开设置页面。
- 设置页面可搜索、选择并修改手动操作过的网站。
- 设置页里的下拉列表最多显示 10 项，更多选项可滚动查看。
- 支持微软翻译、Google 翻译、MyMemory 翻译和自定义翻译接口。
- 会监听页面变化、滚动加载内容，并尝试补翻新出现的文字。

## 安装方法

1. 在浏览器安装用户脚本管理器，例如 Tampermonkey、Violentmonkey 或 Greasemonkey。
2. 新建用户脚本。
3. 将 `自动翻译.txt` 中的全部内容复制进去。
4. 保存脚本并启用。
5. 打开任意网页，右上角会出现“译”图标。

## 基本使用

### 默认翻译

脚本安装启用后，默认会翻译打开的网站。可以在设置页把全局默认行为改为“不翻译”。没有手动操作过的网站不会出现在设置页的网站列表里。

第一次安装后打开网页时，“译”图标周围会短暂闪烁 3 秒，提示先查看默认设置。

### 关闭当前网站翻译

1. 点击页面右上角的“译”图标。
2. 在弹窗中点击“不翻译”。
3. 当前网站会被记住，以后打开这个网站默认不翻译。

### 重新开启当前网站翻译

1. 点击“译”图标。
2. 在弹窗中点击“翻译”。
3. 当前网站会被记住，以后打开这个网站默认翻译。

### 移动图标位置

按住右上角的“译”图标并拖动到想放的位置，松开后位置会自动保存。下次打开网页时会使用上次的位置。

### 图标自动淡化

“译”图标在 5 秒无操作后会自动淡化。选择“翻译”或“不翻译”后，也会立即淡化。默认淡化后不透明度为 50%。

打开弹窗、拖动图标或鼠标悬停在图标上时，图标会恢复清晰显示。

## 设置页面

点击“译”图标后，再点击弹窗右上角的齿轮按钮，可以打开设置页面。

设置页面主要包含这些内容：

### 默认行为

这里可以用开关选择“默认是否自动翻译”。

- 开启：没有手动设置过的网站会自动翻译。
- 关闭：没有手动设置过的网站不会自动翻译，只有手动选为“翻译”的网站才会翻译。

### 翻译服务

默认使用“微软翻译”。可以切换为：

- 微软翻译
- Google 翻译
- MyMemory 翻译
- 自定义接口

切换翻译服务后，脚本会清空当前翻译缓存，并使用新的服务重新翻译后续内容。设置页里的选项会自动保存，不需要点击保存按钮。

### 翻译方向

默认方向是“英语 -> 简体中文”。可以在设置页里选择：

- 来源语言：英语、简体中文、法语等，也可以选“所有语言”自动识别。
- 目标语言：简体中文、法语、英语等具体语言。

示例：

- `英语 -> 简体中文`
- `简体中文 -> 法语`
- `所有语言 -> 简体中文`

选择后会直接改变翻译接口的来源语言和目标语言参数。

### 图标显示

图标显示设置放在“翻译服务”区域里。可以修改“淡化后不透明度”，填写范围建议为 10 到 100：

- `50` 表示淡化后显示为 50% 不透明。
- `100` 表示不淡化。
- 数值越小，图标越透明。

### 手动操作过的网站

这里只显示你手动点过“翻译”或“不翻译”的网站。自动跟随默认行为的网站不会显示在这里。

你可以在这里：

- 点击放大镜展开搜索框，再输入关键字搜索网站。
- 从下拉列表中选择网站。
- 用“是否翻译”开关修改选中网站。

当前版本不再提供单独的“默认”按钮。需要改变某个网站时，直接在下拉列表里选择它并修改翻译选择。

## 自定义翻译接口

在设置页面选择“自定义接口”后，可以填写接口地址和结果字段。

接口地址支持这些占位符：

- `{text}`：要翻译的文本。
- `{from}`：来源语言，无法判断时为 `auto`。
- `{to}`：目标语言，默认是简体中文。

示例：

```text
https://example.com/translate?q={text}&from={from}&to={to}
```

如果接口返回 JSON，可以在“结果字段”里填写翻译结果所在字段，例如：

```text
translatedText
```

也支持简单的点路径：

```text
data.text
```

如果接口返回的不是 JSON，脚本会把接口返回的文本直接当作翻译结果。

## 注意事项

- 翻译内容会发送到你选择的翻译服务，请不要在敏感页面开启翻译。
- 自定义接口目前按 GET 请求调用。
- 某些网站有复杂的动态页面结构，可能会出现少量文字未翻译或重复补翻。
- 如果网站本身禁止脚本运行，油猴脚本可能无法生效。
- MyMemory 等免费接口可能有频率限制，翻译大量文字时速度可能较慢。

## 常见问题

### 为什么有些网站没有出现在设置页？

设置页只显示手动操作过的网站。默认自动翻译的网站不会显示。

### 关闭翻译后还能恢复吗？

可以。点击“译”图标后选择“翻译”，或在设置页中把该网站改为“翻译”。

### 如何回到默认状态？

进入设置页，调整“默认是否自动翻译”开关。已经手动设置过的网站会继续使用自己的选择。

### 更换翻译服务后之前的结果会怎样？

脚本会清空翻译缓存，并使用新服务处理之后的翻译。已经显示在页面上的翻译内容会尽量恢复后再重新翻译。
